ECS七天训练营 Day3

Day 3 基于云服务器构建微信公众号管理系统

运行环境：LAMP环境（Ubuntu16.04 Apache PHP7.1）
购买连接

环境配置与微擎安装

  1. 登录服务器 输入代码 然后按回车下一步
  2. 您想现在安装bt面板到www目录吗？ 输入y 然后按回车下一步
  3. 安装宝塔完毕，服务器会提示宝塔面板的管理登录地址，账号密码（用电脑的浏览器访问登录）
  4. 登录宝塔面板,和图片一样选，然后等候一小时左右即可安装完毕，可以宝塔后台的左上角看到任务安装列表
  5. 第4步安装完成后才能部署，不然无法部署
  6. 下载安装包，http://cdn.w7.cc/download/WeEngine-Laster-Online.zip
    解压出一个php文件，点击网站目录上传此文件。
  7. 访问 你的域名/install.php 安装，这里是s.w7.cc的账户，没有就去注册

    如果弹出这个

    请参考此教程
  • 编辑PHP.INI
  • 找到;always_populate_raw_post_data = -1
  • 去掉前面的分号;如果=号后面是on ，改成-1
  • 保存,重启php
  1. 弹出的内容对应填写
  2. 设置此后台的账户密码 ，安装成功

公众号添加与管理

  1. 更新一下系统，及时更新系统可以避免一些错误
  2. 添加需要管理的微信公众号
  3. 选择【手动添加公众号】
  4. 根据页面要求填写内容

    可以在微信公众号后台的这两个部分中查找所需要的信息。
  5. 有一步需要在微信公众号的后台添加开发者信息，将系统给出的 Token和 Key 复制到【基本配置】中即可




    这样就添加好公众号了，接下来我们测试一下公众号应用。
  6. 进入【应用市场】，挑选一款适合自己的公众号应用
  7. 回到网站，进入【未安装公众号应用】，安装一下刚刚购买的应用


  8. 返回添加好的公众号，就可以看到刚刚安装好的应用了
  9. 点击一个应用就可以进行设置或者扫二维码使用了
